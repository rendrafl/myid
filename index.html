Rendra Felani.

Nama saya Rendra Felani, saya seorang staff IT dengan fokus pada infrastruktur. lahir di Surabaya 23 Mei 1994, saya memiliki minat pada komputer khususnya tentang kinerja komputer dalam bidang game, multimedia dan pemprosesan data.


Sejak lulus dengan S.Kom. ilmu komputer dari Universitas Narotama pada tahun 2018, saya memfokuskan menjadi seorang system administrator dengan mengikuti bidang keilmuan tentang High Performance Computing (HPC), Pararel Processing, Virtualization, Network Administrator serta Manajemen Server.


Saat ini saya bekerja sebagai staff IT infrastruktur di QSoft Surabaya dan berspesialisasi dalam masalah administrasi Server beserta infrastruktur didalamnya.


Selain pekerjaan pengembangannya, saya termasuk anggota dari Tim peneliti dari Universitas Narotama dan telah menerbitkan sebuah paper penelitan tentang container dan neural network (2018).


Ketika sedang libur, saya berada dirumah yang terletak di Masangan kulon, Sidoarjo bersama kedua orang tua, menghabiskan waktu luang dengan belajar, browsing, menonton film, mendengarkan musik dari band favorit saya, dan bereksperimen dengan komputer, atau bermain bersama hewan peliharaan burung merpati.
